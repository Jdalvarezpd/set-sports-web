# SET Sports Web ⚽🏆

> ⚠️ **Nota de Archivo (2018):** Este proyecto fue desarrollado originalmente en el año **2018**. Se ha subido a este repositorio exclusivamente con fines de archivo histórico, para documentar mi trayectoria en el desarrollo web y mi experiencia trabajando con PHP, SQL y JavaScript. Actualmente la plataforma se encuentra inactiva y no recibe mantenimiento.

## 📖 Sobre el Proyecto

**SET** nació como una solución digital diseñada para modernizar y facilitar la organización de torneos de fútbol en la ciudad de Cali. 

La plataforma web funcionaba como el núcleo administrativo donde los organizadores podían crear competiciones, gestionar equipos, programar partidos y llevar un control automatizado de las tablas de posiciones. Su objetivo principal era digitalizar y optimizar un proceso logístico complejo que tradicionalmente se manejaba de forma manual.

*Nota: Este repositorio contiene únicamente la versión web de la plataforma. El ecosistema SET también contaba con una aplicación móvil complementaria.*

## 🚀 Características Principales

* **Gestión de Torneos:** Creación de ligas y campeonatos con parámetros personalizados.
* **Panel Administrativo:** Control centralizado sobre equipos inscritos, registro de jugadores y gestión de sanciones.
* **Sistema de Fixtures:** Programación de calendarios de partidos, asignación de canchas y registro de resultados en tiempo real.
* **Tablas y Estadísticas:** Cálculo y actualización automática de posiciones, puntajes y diferencia de goles.

## 🛠️ Tecnologías Utilizadas

Este proyecto refleja una etapa de desarrollo enfocada en soluciones robustas y a medida utilizando tecnologías web base:

* **Backend:** PHP y SQL para la lógica de negocio y gestión de base de datos relacional.
* **Frontend:** HTML5, CSS3 y JavaScript para la interfaz de administración.
